# Microsoft_engage21

# Running it on your device

* Download the Zip file and extract it

* npm ci

* cd src

* node app.js

# How I worked on this Challenge

I have implemented the video call app using

* webRTC

* Socket.io

* Node.js

 It can handle maximum 4 devices.
